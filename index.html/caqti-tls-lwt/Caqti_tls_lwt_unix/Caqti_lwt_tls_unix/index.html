Module Caqti_tls_lwt_unix.Caqti_lwt_tls_unix

TLS provider for caqti-lwt based on the tls-lwt library.

This sublibrary has no entry point, but registers TLS support for caqti-lwt.unix, which can be enabled by setting Caqti_tls.Config.client in the configuration.
