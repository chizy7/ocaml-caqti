Module Caqti_mirage

Functions for connecting to databases from MirageOS unikernels

This module contains functions for connecting to databases using the MirageOS platform libraries, providing support for PGX but not drivers based on bindings.

See also Caqti_lwt for basic Lwt support.

The caqti-mirage library is experimental at this point. Feedback from MirageOS users on the current API is very welcome.
